Welcome in the fire of the forest game.

It's game based on cellular automation and Moore neighborhood. 

Inside we can see simulation of the forest during fire and user can put firefighters to put the fire off.

User has two game modes:
- normal mode
    There are three levels and user needs to put the firefigters in order to stop the fire. 
    If more than 50% if the forest is still alive, user wins.
- free mode
    User can change variables and put firefighters and lightning when and where he wants.
